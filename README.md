# Discord Ticket Bot 🎫

Bot de tickets para Discord com sistema completo de suporte.

## 🚀 Funcionalidades
- ✅ Sistema de tickets com 3 tipos
- ✅ Botão fixo para abrir tickets
- ✅ Controles para staff
- ✅ Sistema de logs completo
- ✅ Hospedagem automática no Railway

## ⚙️ Configuração
1. Clone este repositório
2. Configure as variáveis de ambiente no `.env`
3. Deploy automático no Railway

## 📝 Comandos
- `/setup-ticket` - Configura o sistema de tickets
